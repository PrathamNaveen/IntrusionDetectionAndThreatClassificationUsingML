Intrustion Detection System Using Machine Learning

This is my 6th Semester Mini-Project in JSSSTU, The project is a collab between the fields of Machine Learning and Cybersecurity.

# Dataset: CICIDS2017

# PURPOSE:
The purpose was to automate the Intrusion detection on Real Time Data so that proper action can be taken to safegaurd against the attacks.

# Tools and Tech Used:
Python, Tensorflow, Numpy, Pandas, Matplotlib, Seaborn, LabelEncoder, Robust Scaler, Wireshark, Scapy

# Steps to run the Project:
If the model is not trained uncomment all the function calls and run the main.py
else run all the calls starting from load_dataset_and_model()
